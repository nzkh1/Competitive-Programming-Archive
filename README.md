## 📅 The Roadmap

### 🟢 Grade 10: Foundations & Efficiency
*Focus: Mastering complexity and standard paradigms.*
- [x] **Frequency Arrays & Vectors:** Optimized counting and memory management.
- [x] **Greedy & Binary Search:** Efficiently narrowing solution spaces.
- [ ] **Prefix Sums & 2D Partial Sums:** $O(1)$ range queries.
- [ ] **Elementary DP:** Introduction to recurrence relations.

### 🟡 Grade 11: Advanced Data Structures
*Focus: Logarithmic complexity and graph theory.*
- [ ] **Graph Algorithms:** Dijkstra, Bellman-Ford, and Minimum Spanning Trees (MST).
- [ ] **Segment Trees & Fenwick Trees:** Dynamic range updates and queries.
- [ ] **Lowest Common Ancestor (LCA):** Tree-based optimizations.
- [ ] **Disjoint Set Union (DSU):** Fast connectivity checking.

### 🔴 Grade 12: Mastery & Specialization
*Focus: Maximum flow, string processing, and geometry.*
- [ ] **Network Flow:** Max Flow / Min Cut (Edmonds-Karp/Dinic).
- [ ] **String Algorithms:** KMP, Z-Algorithm, and Suffix Arrays.
- [ ] **Computational Geometry:** Convex Hull and sweep-line algorithms.
- [ ] **Advanced DP:** Convex Hull Trick (CHT) and DP on trees.

## 💻 Development Environment

I primarily develop locally, but I use online compilers for quick testing and debugging.

*   **Primary Editor:** [Visual Studio Code](https://code.visualstudio.com)
    *   **Extensions:** `ms-vscode.cpptools` (C/C++), `formulahendry.code-runner`
*   **Compiler:** `g++` (MinGW-w64) using the `-O3` optimization flag.
*   **Backup IDE:** [OnlineGDB](https://www.onlinegdb.com) — Used for quick verification and remote practice.
